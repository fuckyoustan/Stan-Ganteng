## Overview
**ANU (Acceleration N Utility)** ain’t your average tweak—it’s a full-blown **SurfaceFlinger dominator**. This Magisk module takes your Android’s rendering system and throws it into overdrive. It is built for those who refuse to settle for stock settings and want to see their device perform at its absolute peak.

---

## What Does It Do?
ANU brings a powerful set of optimizations to the table, targeting the core of Android's visual performance:

- **⚡ SurfaceFlinger Boost:** Overhauls the composition engine for maximum throughput.
- **🖼️ Frame Timing Optimization:** Ensures consistent frame delivery to eliminate stutters.
- **🎨 HWUI Rendering Tweaks:** Accelerates hardware-accelerated UI tasks.
- **🖱️ Touch Input Enhancement:** Reduces input latency for a more responsive feel.
- **🚀 Animations Turbo:** Speeds up system animations without breaking visual integrity.
- **💎 ART Runtime Boost:** Optimizes the Android Runtime for faster app execution.
- **🔄 VSYNC & Refresh Rate Control:** Fine-tunes sync points for ultra-smooth scrolling.
