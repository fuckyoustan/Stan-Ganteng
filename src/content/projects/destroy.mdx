## Key Features
This module’s script performs the following thermal-obliterating tasks:

- ❌ **Kill Thermal Services:** Completely shuts down system-level thermal management services.
- 🧨 **Property Reset:** Resets all thermal-related system properties to prevent frequency capping.
- 🥶 **Disable Zones:** Disables all thermal zones and cooling device limits.
- 🔓 **GPU Unlock:** Forces the GPU to stay in high-performance states.
- 🧠 **Max Frequencies:** Maxes out CPU frequencies across all clusters.
- 🛡️ **Bypass Protection:** Disables core thermal protections and overrides TZCPU configurations.
